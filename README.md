# DSA Practice

This repository contains solutions to various Data Structures and Algorithms problems. All code is written in Java and organized by topic.

- Arrays
- Strings
- Linked Lists
- Stacks and Queues
- Trees and Graphs
- Recursion and Backtracking
- Dynamic Programming
- Searching and Sorting
